# BrooklynBridgePoker  
  
JavaFX project for basic poker game (5 cards in hand)  
  
OPTIONS MENU  

		-Name - nickname for the player  
		-CPU players - number of CPU bots  
		-Difficulty (normal and cheater)  
          #normal - face down cards  
          #cheater - face up cards
